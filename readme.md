# Student Performance Prediction – Machine Learning Project

## Objective
To predict student final grades using regression and classify pass/fail status using machine learning techniques.

## Techniques Used
- Exploratory Data Analysis
- Data Preprocessing
- Linear & Polynomial Regression
- Logistic Regression, KNN, Decision Tree
- Model Evaluation Metrics

## Results
- Polynomial Regression achieved highest R² score
- Logistic Regression performed best for classification

## Conclusion
The project demonstrates an end-to-end ML pipeline and highlights the importance of preprocessing and model selection.
